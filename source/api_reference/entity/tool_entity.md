# `ToolEntity`

A kinematically-scripted collider created from a {py:class}`gs.materials.Tool <genesis.engine.materials.tool.Tool>` material. Its motion is prescribed rather than dynamically simulated, and it couples one-way with soft and particle-based media to push, stir, or deform them.
