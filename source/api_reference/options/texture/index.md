# Texture

## 概述

纹理 (Texture) 是 Genesis 引擎中用于增强材质视觉效果的重要组件。通过纹理，可以为表面添加颜色变化、细节、法线信息和其他视觉特性，使渲染结果更加真实和丰富。纹理系统支持多种纹理类型，包括基本颜色纹理和图像纹理，可与各种材质类型结合使用。

## 主要特点

- **多种纹理类型**：支持颜色纹理和图像纹理等多种纹理格式
- **灵活的应用方式**：可应用于材质的颜色、粗糙度、法线等多个属性
- **高质量渲染**：支持高分辨率纹理和各种纹理映射技术
- **与材质系统无缝集成**：可与塑料、金属、玻璃等材质类型完美配合
- **纹理坐标控制**：支持纹理坐标变换和映射方式调整
