# 路线图

## 进行中，即将在近期发布
- 可微分 & 基于物理的触觉传感器模块
- 可微分刚体仿真
- 分块渲染 (Tiled rendering)
- 更快的 JIT 核函数编译
- 综合性生成框架
    - 角色动作 (character motion)
    - 相机运动 (camera motion)
    - 交互式场景
    - 面部动画
    - 运动策略 (locomotion policy)
    - 操作策略 (manipulation policy)
- 无边界 MPM 仿真，用于大规模环境

## 期望功能，但目前尚未开始开发（欢迎贡献！）
- Windows 平台上的查看器 (Viewer) 和无头渲染 (headless rendering)
- 交互式 GUI 系统
- 支持更多基于 MPM 的材料模型
- 更多传感器类型
